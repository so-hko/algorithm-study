#체육복 문제 솔루션
- solution 1

```python

def solution(n, lost, reserve):
    tmp = [x for x in lost if x not in set(reserve)]
    reserve = [x for x in reserve if x not in set(lost)]
    lost = tmp
    lost.sort()
    reserve.sort()
    for i in reserve : 
        for j in lost : 
            if abs(i-j)==1 :
                lost.remove(j)
                break
    answer = n-len(lost)
    return answer

```
